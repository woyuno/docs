## VoidChain Overview

VoidChain (Trust node blockchain network) technical architecture represents a historic moment in the development of blockchain. The blockchain mainnet network is a standard privacy computing service. In the early days of blockchain invention, it was unimaginable to create its own privacy computing network, and running privacy computing services on a public network also led to one of the four fundamental technologies of blockchain - Byzantine Fault Tolerance technology. The other three fundamental technologies are mature technologies invented early in history. The later development history of blockchain became the development history of Byzantine Fault Tolerance technology.

VoidChain brings a trusted privacy computing network. The entire mainnet nodes operate in a decentralized network vacuum that is immune to human interference. The network is completely isolated from the outside, with the only communication channel being through self-verifiable edge trust machines via encrypted channels. The VoidChain mainnet architecture fills the technical gap in blockchain, allowing pure privacy computing services to return to a privacy computing environment, achieving the complete implementation of blockchain technology.
